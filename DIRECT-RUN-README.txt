AZEZ LAB AI — DIRECT RUN PACKAGE

WINDOWS:
1. Extract the ZIP.
2. Double-click START-AZEZ-LAB-AI.bat
3. The browser opens automatically.

The first run installs npm dependencies if node_modules is not present, so internet is required for the first dependency installation.
After dependencies are installed, later launches use the existing node_modules folder.

STOP:
Double-click STOP-AZEZ-LAB-AI.bat

REQUIREMENT:
Node.js must already be installed and available in PATH.

LOCAL ADDRESSES:
UI   http://127.0.0.1:5173
CORE http://127.0.0.1:8787/api/health

IMPORTANT:
This remains an engineering/clinical decision-support prototype. Do not use it for real patient diagnosis or unsupervised clinical decisions.
