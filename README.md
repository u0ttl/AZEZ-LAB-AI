# AZEZ LAB AI V16 — Validation & Automated Testing

Run `START-AZEZ-LAB-AI.bat`.

After dependencies are installed:

`npm run test:core`

Then:

`npm run validate:release`

V16 measures extraction precision/recall/F1, verification routing rate, and AI output rejection rate. The release validator runs document parsing, laboratory knowledge, tenant isolation, AI governance, and validation-metrics suites.

Engineering validation is not clinical validation. Clinical efficacy must be established separately using authorized representative datasets, defined protocols, qualified reviewers, and the requirements of the intended deployment market.
